# Spotify Landing Page

Projeto acadêmico de recriação da landing page do Spotify, desenvolvido como estudo de front-end com foco em HTML, CSS e Bootstrap.

## Tecnologias

- HTML5
- CSS3
- Bootstrap 3
- jQuery

## Seções

- **Hero** — chamada principal "Música para todos" com botões de CTA
- **Serviços** — destaques do Spotify (músicas, playlists, lançamentos)
- **Recursos** — funcionalidades do app (buscar, navegar, descobrir)
- **Rodapé** — links institucionais e redes sociais
